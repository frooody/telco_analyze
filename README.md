# Анализ оттока клиентов телеком-компании

## Описание проекта
Данный проект посвящён анализу клиентской базы телеком-компании с целью выявления факторов, влияющих на отток клиентов (Churn), а также проверке бизнес-гипотез на основе имеющихся данных.

Анализ выполнен в формате исследовательского ноутбука с использованием Python и библиотек для обработки данных и визуализации.

---

## Цель работы
Выявить ключевые факторы, влияющие на отток клиентов, и проверить ряд бизнес-гипотез для последующего улучшения удержания пользователей.

---

## Задачи проекта
- Провести первичный анализ и очистку данных
- Привести данные к удобному формату (типизация, обработка пропусков)
- Выполнить исследовательский анализ (EDA)
- Проверить сформулированные гипотезы
- Определить закономерности поведения клиентов
- Сформировать портрет лояльного клиента
- Сделать выводы и предложить рекомендации

---

## Используемые данные
Датасет содержит информацию о клиентах телеком-компании, включая:
- демографические характеристики (пол, возраст, наличие партнёра и иждивенцев)
- используемые услуги (интернет, безопасность, ТВ и др.)
- параметры контракта (тип, длительность, стоимость)
- факт оттока (Churn)

---

## Основные этапы работы

### 1. Загрузка и предобработка данных
- Загрузка датасета
- Очистка пропусков
- Приведение типов данных
- Переименование колонок

### 2. Исследовательский анализ данных (EDA)
- Анализ распределений признаков
- Проверка на дубликаты
- Описательная статистика

### 3. Feature Engineering
- Создание новых признаков
- Преобразование категориальных переменных

### 4. Проверка гипотез
В работе проверяются следующие гипотезы:

1. Клиенты с партнёром чаще подключают дополнительные услуги  
2. Пользователи оптоволоконного интернета реже подключают защитные сервисы  
3. Пожилые клиенты чаще выбирают долгосрочные контракты  
4. Долгосрочные клиенты платят меньше  
5. Клиенты с малым сроком обслуживания чаще уходят  
6. Клиенты с иждивенцами более лояльны  

### 5. Анализ оттока
- Выявление групп с высоким churn
- Анализ поведения клиентов
- Сравнение сегментов пользователей

### 6. Формирование выводов
- Определение ключевых факторов оттока
- Построение портрета клиента

---

## Основные результаты
- Наибольший отток происходит в первые месяцы использования
- Клиенты с краткосрочными контрактами уходят чаще
- Дополнительные сервисы снижают вероятность оттока
- Пожилые клиенты демонстрируют более высокий churn
- Лояльные клиенты — это пользователи с длительным сроком обслуживания

---

## Практическая значимость
Результаты анализа могут быть использованы для:
- разработки стратегий удержания клиентов
- оптимизации тарифов
- повышения вовлечённости пользователей
